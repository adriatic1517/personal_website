Hello.

This is a Pashto to English dictionary written about 200 years ago and digitized by the University of Chicago's Digital Dictionaries of South Asia project.

This is the beginning of what I hope to be a broader project of digitizing Pashto language and literature.

Some features that this project adds on are a very fast, scalable search algorithm, responsive interface and accessibility to the general public.

I created it with the help of these kind people.

If you would like to know more about this project, read my blog post (coming soon) on it.
